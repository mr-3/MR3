ファイル

ファイル -> 新規
MR3のプロジェクトを新規に開く． 現在の作成中のプロジェクトは，保存するか破棄するかを 選択する．
ファイル -> 開く
MR3のプロジェクトファイルを開く．
ファイル -> 保存
MR3のプロジェクトファイルを保存する．
ファイル -> 名前をつけて保存
MR3のプロジェクトファイルを別名で保存する．
ファイル -> インポート -> 置き換え
ファイルまたは，URIから，RDF/XMLまたは，RDF/N-Triple形式で記述されたのモデルを インポートする．RDFエディタで構築中のモデルは，インポートとしたモデルに置き換えられる．
ファイル -> インポート -> 置き換え -> RDF
RDF/XML (File)
RDF/XML形式のファイルをインポートする．
RDF/XML (URI)
URIを入力して，RDF/XML形式で記述されたモデルをインポートする．
RDF/N-Triple (File)
RDF/N-Triple形式のファイルをインポートする．
RDF/N-Triple (URI)
URIを入力して，RDF/N-Triple形式で記述されたモデルをインポートする．
ファイル -> インポート -> 置き換え -> RDFS
RDFS/XML (File)
RDFS/XML形式のファイルをインポートする．
RDFS/XML (URI)
URIを入力して，RDFS/XML形式で記述されたモデルをインポートする．
RDFS/N-Triple (File)
RDFS/N-Triple形式のファイルをインポートする．
RDFS/N-Triple (URI)
URIを入力して，RDFS/N-Triple形式で記述されたモデルをインポートする．
ファイル -> インポート ->マージ
ファイルまたは，URIから，RDF/XMLまたはRDF/N-Triple形式のモデルをインポートする． RDFエディタで構築中のモデルに，インポートしたモデルをマージする．
RDF(S)/XML (File)
RDF(S)/XML形式のファイルをインポートする．
RDF(S)/XML (URI)
URIを入力して，RDF/XML形式で記述されたモデルをインポートする．
RDF(S)/N-Triple (File)
RDF(S)/N-Triple形式のファイルをインポートする．
RDF(S)/N-Triple (URI)
URIを入力して，RDF/N-Triple形式で記述されたモデルをインポートする．
ファイル -> インポート -> プロジェクト (Javaのオブジェクト)
Javaのオブジェクトの形式で保存されたMR3のプロジェクトを， インポートする．
ファイル -> エクスポート -> RDF/XML
MR3で構築したRDFモデルまたは，RDFSモデルをエクスポートする．
RDF/XML
RDFエディタで構築したモデルをRDF/XML形式でファイルに保存する．
RDFS/XML
クラスエディタ及びプロパティエディタで構築したモデルを RDF/XML形式でファイルに保存する．
選択されているRDF/XML
RDFエディタで構築したモデルの中から，選択されたノードを RDF/XML形式でファイルに保存する．
選択されているRDFS/XML
クラスエディタ及びプロパティエディタで構築したモデルの中から，選択されたノードを RDF/XML形式でファイルに保存する．
ファイル -> エクスポート -> N-Triple
RDF/N-Triple
RDFエディタで構築したモデルをN-Triple形式でファイルに保存する．
RDFS/N-Triple
クラスエディタ及びプロパティエディタで構築したモデルを N-Triple形式でファイルに保存する．
選択されている RDF/N-Triple
RDFエディタで構築したモデルの中から，選択されたノードを N-Triple形式でファイルに保存する．
選択されている RDFS/N-Triple
クラスエディタ及びプロパティエディタで構築したモデルの中から，選択されたノードを N-Triple形式でファイルに保存する．
ファイル -> エクスポート -> Image
RDF Graph -> PNG
RDFグラフをPNG形式でファイルに保存する．
Class Graph -> PNG
クラスグラフをPNG形式でファイルに保存する．
Property Graph -> PNG
プロパティグラフをPNG形式でファイルに保存する．
ファイル -> エクスポート -> プロジェクト (Javaのオブジェクト)
Javaのオブジェクトの形式でMR3のプロジェクトを エクスポートする．
ファイル -> プラグイン
プラグインのメニューが表示される．
ファイル -> 終了
プログラムを終了する．
