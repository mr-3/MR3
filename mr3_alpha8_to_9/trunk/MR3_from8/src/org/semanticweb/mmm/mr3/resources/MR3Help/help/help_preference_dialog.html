設定ダイアログ

設定(基本タブ)

設定ダイアログの基本タブでは，デフォルトの言語，UI言語，入出力ファイ ルのエンコーディング，BaseURI，Work Directory，プロキシの設定ができる． デフォルトの言語は，ラベル表示を行う際に優先して表示する言語を指定できる． UI言語は，メニュー等の表示の言語を選択できる．現在のところ，"ja"(日本語)と "en"(英語)のどちらかを選択できる．UI言語の変更は，次回起動時から有効になる． BaseURIには，export時にIDとしたいURIの名前空間を指定する．Work Directoryには，importファイル選択時にはじめに開くフォルダを指定する．



図18: 設定 (基本タブ)

設定(メタクラスリストタブ)

設定ダイアログのメタクラスリストタブでは，どのリソース(URI)をタイプと するリソースを，RDFSクラスまたは，RDFSプロパティとみなすかを設定する． デフォルトでは，http://www.w3.org/2000/01/rdf-schema#Classをタイプとす るリソースをRDFSクラスとみなしている．また， http://www.w3.org/1999/02/22-rdf-syntax-ns#Propertyをタイプとするリソー スをRDFSプロパティとみなしている．バージョンの異なるrdfs:Classや rdf:Property をタイプとするリソースをRDFSクラス，RDFSプロパティとみな したい場合や，rdfs:Classのサブクラスやrdf:Propertyのサブクラスをタイプ とするリソースをRDFSクラス，RDFSプロパティとみなしたい場合には，Meta Class Listタブで設定を行う．Class Class Listに追加したURIをタイプとす るリソースは，RDFSクラスとみなされる．Property Class Listに追加したURI をタイプとするリソースは，RDFSプロパティとみなされる．

設定(レンダリングタブ)

設定ダイアログのレンダリングタブでは，RDFリソース，RDFリテラル，RDFSク ラス，RDFSプロパティ，選択時，エディタウィンドウの背景の，それぞれの色 をカスタマイズすることができる．Colorチェックボックスのチェックをはず すと，すべて無色となる．AntialiasチェックボックスをチェックするとRDFリ ソースやRDFリテラル等の楕円，矩形の縁がなめらかになる．
